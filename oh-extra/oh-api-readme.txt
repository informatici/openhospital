######
oh-api.sh (Linux) and oh-api.ps1 (Windows) are two scripts used to test the full OH distribution: Core + GUI + API + UI.

- on Linux, launch the script with ./oh-api.sh
- on Windows, launch the script with powershell.exe oh-api.ps1
- select SERVER mode to run OH API server.

Dashboard is available by opening the web address:

http://localhost:8080
