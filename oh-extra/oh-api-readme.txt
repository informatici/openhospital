To run the full OH distribution Core + GUI + API server + UI, use the bundled oh.sh / oh.ps1 script:

- on Linux, launch the script with ./oh.sh
- on Windows, launch the script with powershell.exe oh.ps1 or command oh.bat
- Select "E" for EXPERT MODE
- Select "A" to activate API SERVER at startup

Dashboard is available by opening the web address:

http://localhost:8080
